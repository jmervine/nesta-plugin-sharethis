# Nesta::Plugin::Sharethis

Add ShareThis to NestaCMS.

## Installation

Add this line to your application's Gemfile:

    gem 'nesta-plugin-sharethis'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install nesta-plugin-sharethis
